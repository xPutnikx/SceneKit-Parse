#Instructions

#Installation
For Run project you just need to unpack it and open Nodes3d.xcworkspace file by XCode
On this project I used CocoaPods for Parse.com
you need to run command in your terminal 
 $ pod install
for install Parse sdk
after that, you can open and run project.
This project configured for my Parse.com application

#Video
Also, please, watch the demo video where explained how does it works.

#Usage
On the application you can open/close nodes with childs by tap
And you can add new node by long press
If you can open node, it will has blue color, after press on it, node will expand
If you can close node, it will has red color, after press on it, node will collapse
By long press you can add node to any level on hierarchy.
